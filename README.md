The goal of vbzdelays is to analyze VBZ delays.

## Installation

You can install vbzdelays with:

``` r
remotes::install_github("miraisolutions/vbzdelays")
```

## Example

Load line data

``` r
vbzdelays::get_line_data(line = 11)
```
